# GAN Design Eval Rubric — VedaAI

### Design Quality (weight: 0.35)
- Does it match the reference screenshots pixel-perfectly?
- Is the orange brand color (#E8531D) consistently applied?
- Is the light/white theme implemented correctly?
- Does the sidebar match the reference (logo, nav items, bottom profile)?
- Are cards clean white with proper borders and shadows?

### Originality (weight: 0.30)
- Does the implementation feel premium and polished?
- Are micro-animations present (hover states, transitions)?
- Is the empty state illustration compelling?
- Does the form feel intuitive and modern?
- Are the question type rows dynamic and well-designed?

### Craft (weight: 0.25)
- Is the code clean, typed, and maintainable?
- Are components properly decomposed?
- Is the CSS/Tailwind usage correct for light theme?
- Do all interactive states (hover, active, focus) work?
- Is it responsive (sidebar → tab bar on mobile)?

### Functionality (weight: 0.10)
- Does the empty state show when no assignments exist?
- Do the assignment cards render with correct data?
- Does the create form still submit correctly?
- Does navigation work between all routes?
